Temperature Converter

  • Category: Web Development
  • Type: Academic
  • Project date: January 2021
  • Used: PHP, HTML, CSS, Bootstrap

The Goal:

Build and style a fully functional temperature converter that will convert to and from Fahrenheit, Celsius and Kelvin units.

The Process and Result:

Using PHP, HTML, CSS and Bootstrap, I created a fully responsive web application that will accurately convert temperatures. It was given a circus/carnival styling to represent the type of variety I aniticipate encountering.
