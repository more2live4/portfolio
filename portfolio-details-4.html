Original GIF Creation

  • Category: Graphic Design
  • Type: Academic
  • Project date: January 2020
  • Used: Adobe Photoshop

The Goal:

Create a working GIF based on a photo or video inside Adobe Photoshop.

The Process and Result:

Using a reference photo from a copyright-free resource, I created a Graphics Interchange Format image. The photo I chose did such a great job of capturing that moment in the sun that I strived to maintain that feeling by creating a natural-ish flight pattern for the birds. I did this by creating twenty-four individual layers, each rotated by fifteen degrees. I slighly altered the positioning of each bird each time. Then using the Motion panel, I timed them out to create a continuous motion.
